 # Drag and Drop Image Project

This is a simple drag and drop project built using HTML, CSS, and JavaScript.

## Project Overview

The project contains four boxes on the screen.  
One box contains an image, and the image can be dragged and dropped into another box.

This project demonstrates:

- Drag and Drop API in JavaScript
- DOM manipulation
- Event handling
- Basic UI layout using CSS

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Drag image from one box
- Drop image into another box
- Interactive user experience
- Clean and simple layout

## Project Structure

```
index.html
style.css
script.js
```

## Purpose

This project was created to practice JavaScript drag and drop functionality and improve frontend development skills.
